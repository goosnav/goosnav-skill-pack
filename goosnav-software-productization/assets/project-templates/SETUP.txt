END-USER LOCAL EDITION
----------------------
M1a Zip Edition: extract, double-click the launcher, and the first-run wizard sets up everything automatically — no manual installs, no terminal commands. Document platform-specific launch and provider-key setup when M1a is complete.
M1b Packaged Edition: the installer bundles the runtime entirely; users never need Python, Node.js, pip, npm, Docker, or build tools.

CONFIGURATION AND SECRETS
-------------------------
- Repo `.env` is for source development only and must not be committed.
- Local edition secrets live in a permission-restricted config store in the app-data directory, edited from the GUI; the OS credential store is preferred and required from M1b.
- Hosted secrets use the deployment platform's secret manager.
